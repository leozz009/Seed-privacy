# Seed-privacy

En Seed valoramos la privacidad de nuestros usuarios. La aplicación no recopila, almacena ni deposita ningún dato del usuario en servidores externos. Toda la información se gestiona exclusivamente de manera local en el dispositivo.

Para ello, Seed utiliza SwiftData, un framework desarrollado por Apple que permite almacenar y manejar datos de forma segura directamente en el dispositivo del usuario. Esto significa que los datos nunca salen del dispositivo y permanecen bajo el control total del usuario en todo momento.

Nuestro compromiso es ofrecer una experiencia segura, privada y transparente, garantizando que tu información personal siempre permanezca contigo.
